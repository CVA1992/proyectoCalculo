Máximos y mínimos

Los puntos máximos y mínimos de una función son valores críticos donde la función cambia su comportamiento (crecimiento/decrecimiento). GeoGebra permite identificarlos visual y analíticamente.

Actividad 1: Identificación de extremos

  1. Ingresa una función polinómica:
    f(x) = x^3 - 3x^2 + 2
  2. Encuentra los puntos críticos:
    Extremos = PuntoRaíz(Derivada(f))
  3. Clasifica cada punto crítico:
    DerivadaSegunda(f, x(Extremos))

    Si f''(a) > 0 → mínimo local; si f''(a) < 0 → máximo local

  4. Visualiza los puntos en la gráfica
Comandos útiles
  • Extremo(f, a, b) - Extremos en [a,b]
  • PuntoRaíz(f') - Puntos críticos
  • DerivadaSegunda(f) - Segunda derivada

Problemas de optimización

La optimización consiste en encontrar valores máximos o mínimos de una función bajo ciertas restricciones. Sigue estos pasos generales:

Ejemplo: Caja de máximo volumen

De una lámina cuadrada de 20 cm de lado, se cortan cuadrados en las esquinas para formar una caja. ¿Qué tamaño maximiza el volumen?

  1. Define la función de volumen:
    V(x) = x*(20-2x)^2
  2. Encuentra el dominio (0 < x < 10)
  3. Calcula los puntos críticos:
    soluciones = PuntoRaíz(Derivada(V))
  4. Evalúa en los extremos y puntos críticos:
    Valor(V, {0, 10, x(soluciones)})
  5. Interpreta los resultados
Tip: Usa Slider para visualizar cómo cambia la caja:
a = Slider(0, 10, 0.1)
Polígono((a,a), (20-a,a), (20-a,20-a), (a,20-a))

Razones de cambio relacionadas

Estudian cómo cambian distintas variables relacionadas entre sí a través del tiempo.

Si y = f(x) entonces dy/dt = f'(x) * dx/dt

Actividad: Globo que se infla

Un globo esférico se infla a razón de 100 cm³/s. ¿Cómo cambia el radio cuando r=5 cm?

  1. Fórmula de volumen:
    V(r) = (4/3)πr³
  2. Deriva implícitamente respecto al tiempo:
    dV/dt = 4πr² dr/dt
  3. Sustituye valores conocidos:
    100 = 4π(5)² dr/dt
  4. Resuelve para dr/dt
Simulación en GeoGebra
  1. Crea un control deslizante para el tiempo:
    t = Slider(0, 10, 0.1)
  2. Define el radio en función del tiempo:
    r(t) = RaízCúbica(3*100*t/(4*π))
  3. Anima el control deslizante
